Profile Matching Algorithm

5 Langkah Algoritma
Profile Matching

Tidak ada kotak hitam. Setiap kalkulasi terbuka, terstruktur, dan dapat dipertanggungjawabkan secara akademis.

1

Vektor Profil Siswa

Input nilai rapor dan skor kuesioner dikemas menjadi vektor [C1..C6]

2

Matriks GAP

GAP = Nilai Siswa - Nilai Target Prodi, dihitung per kriteria per alternatif.

3

Pembobotan GAP

Setiap nilai GAP dikonversi ke bobot standar: GAP=0 menjadi 5.0, GAP=+1 menjadi 4.5, dst.

4

Core Factor (60%) + Secondary Factor (40%)

N = (0.6 x NCF) + (0.4 x NSF) per aspek akademik dan bakat.

5

Total Nilai Akhir & Ranking

Total = (50% x N_Akademik) + (50% x N_Bakat). Diurutkan secara descending.
